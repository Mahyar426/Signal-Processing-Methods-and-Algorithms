This repository contains laboratory exercises, notes, and practical implementations from the Signal Processing: Methods and Algorithms course at Politecnico di Torino. It covers essential signal processing concepts, random processes, estimation methods, spectral analysis, and system modeling.

Repository Contents

Laboratory 2: Nonstationary Random Processes

Generation of random processes with time-varying mean and variance

Simulation and analysis of nonstationary random processes

Laboratory 3

Simulation and analysis of Wiener processes

Langevin equation and Ornstein-Uhlenbeck processes

Laboratory 4

Estimation of mean, variance, and autocorrelation of Wide Sense Stationary (WSS) processes

Sliding window estimation techniques for nonstationary processes

Laboratory 5

Analysis of atomic clock data: Sliding estimations of mean and standard deviation

Spectral estimation using periodograms (basic and Welch methods)

Laboratory 6

Computation and analysis of spectrograms

Time-frequency analysis of stationary and nonstationary signals (e.g., chirp signals)

Laboratory 7

Simulation of discrete-time random systems (boat motion model)

Kalman filtering for state estimation

Laboratory 8

Theoretical and simulated analysis of detector performances

Receiver Operating Characteristic (ROC) curves

Hypothesis testing and performance analysis

Learning Objectives

Understand fundamental concepts in signal processing, random processes, and system analysis

Gain hands-on experience with simulation and estimation algorithms in MATLAB

Apply techniques such as Kalman filtering and spectral estimation practically

Analyze performance of detection systems using statistical tools

Tools and Software

MATLAB for simulations, statistical analysis, and visualizations
